# Ερμηνευτικό Λεξικό Όρων (Glossary)

```{glossary}


Ανάστροφος Πίνακας
  Ο *ανάστροφος πίνακας* (transposed table) $Β$ ενός πίνακα $Α$ είναι ο πίνακας
  του οποίου οι γραμμές είναι οι στήλες του $Α$ και του οποίου οι στήλες είναι
  οι γραμμές του $Α$.

Σετ Δεδομένων
  Το *σετ δεδομένων* (ή *σύνολο δεδομένων*, dataset) είναι μία συλλογή τιμών
  (values), συνήθως αριθμητικών (αν τα δεδομένα αφορούν σε ποσοτικές μετρήσεις)
  ή χαρακτήρων (strings, αν τα δεδομένα αφορούν σε ποιοτικές μετρήσεις).

Data Structure
  Η *δομή δεδομένων* (data structure) είναι μία "οντότητα" (δομή, κατασκευή,
  έννοια, περιέκτης κ.λπ.) στην οποία αποθηκεύονται και οργανώνονται δεδομένα.
  Είναι, γενικά, ένας τρόπος τακτοποίησης δεδομένων σε ένα υπολογιστή, ώστε να
  είναι δυνατή η πρόσβαση σε αυτά και η αποτελεσματική ενημέρωσή τους.

Iterable
  (Python) Κάθε αντικείμενο στο οποίο μπορεί να εφαρμοστεί κάποιος βρόγχος
  επανάληψης (*for* loop, *while* loop κ.λπ.). Τέτοια αντικείμενα είναι οι
  λίστες, τα λεξικά, οι συμβολοσειρές κ.λπ.

```
